## Description
Create a Homebrew tap at `github.com/yotam180/homebrew-pi` and wire GoReleaser to auto-update the formula on every release. After this task, `brew install yotam180/pi/pi` works on macOS for both Intel and Apple Silicon.

## Acceptance Criteria
- [ ] `github.com/yotam180/homebrew-pi` repo exists and is public
- [ ] A formula file `Formula/pi.rb` exists in the tap repo (can be auto-generated by GoReleaser on first release)
- [ ] `brew tap yotam180/pi && brew install pi` installs a working binary
- [ ] `brew upgrade pi` works after a subsequent release
- [ ] GoReleaser's `brews:` config section commits the updated formula to `homebrew-pi` on each release
- [ ] Both `darwin/amd64` and `darwin/arm64` are covered (Homebrew handles the arch selection)
- [ ] The formula sets the correct binary name (`pi`) and passes `brew test`
