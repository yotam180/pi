## Description
Create a Homebrew tap at `github.com/yotam180/homebrew-pi` and wire GoReleaser to auto-update the cask on every release. After this task, `brew install yotam180/pi/pi` works on macOS for both Intel and Apple Silicon (once the `pi` repo is public).

## Implementation Notes

### Decision: homebrew_casks vs brews
GoReleaser v2.10+ deprecated `brews:` (Homebrew Formulas) in favor of `homebrew_casks:`. The old approach used Formulas to install pre-built binaries, which was a misuse of the Formula concept. Casks are the correct mechanism for distributing pre-built binaries.

Key differences from the original plan:
- Used `homebrew_casks:` instead of `brews:`
- Directory is `Casks/` instead of `Formula/`
- Added `postflight` hook to remove macOS quarantine bit (since we don't code-sign)
- `skip_upload: auto` prevents pre-release tags from updating the tap
- No `test:` stanza (not supported in homebrew_casks)

### CI test fixes (bonus)
Discovered and fixed pre-existing CI test failures:
1. Setup tests (`internal/cli/setup_test.go`) failed in CI because they didn't clear CI env vars — added `clearCIEnvVars()` helper
2. Integration shell tests (`tests/integration/shell_test.go`) failed similarly — added `runPiWithHomeNoCI()` helper that strips CI env vars from the subprocess environment
3. Release workflow was missing Node.js + tsx setup — added `actions/setup-node@v4` + `npm install -g tsx` matching the CI workflow

### Release v0.1.0
- Tagged and released successfully via GitHub Actions
- GoReleaser produced 4 archives: darwin/amd64, darwin/arm64, linux/amd64, linux/arm64
- Cask `Casks/pi.rb` pushed to `yotam180/homebrew-pi` automatically
- `brew tap yotam180/pi` works (taps 1 cask)
- `brew install yotam180/pi/pi` fails with 404 because `yotam180/pi` is a private repo

### Remaining blocker
The `pi` repo is private. GitHub release assets from private repos require authentication to download. Homebrew doesn't support authenticated downloads natively. Resolution: make the repo public when ready for distribution.
